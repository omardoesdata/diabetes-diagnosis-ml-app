# Medical Diagnosis ML System — Diabetes Prediction

This is an end-to-end data science project that predicts diabetes using:

-  SQL (Data storage & queries)
-  Machine Learning (Logistic Regression, XGBoost)
-  Deep Learning (Neural Network with TensorFlow/Keras)
-  Streamlit Web App for user input & live predictions
-  Data Visualization + Feature Importance
